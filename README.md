## Installation

1. Open Extensions on the sidebar.
2. Search for Ruie Dark.
3. Install it.
4. Reload ir.
5. `⌘ + ⇧ + p` and type "Color Theme".
6. Pick Ruie Dark from the list and hit enter.
7. Done.

**Enjoy!**
